# S9.3 Retrospective: Secrets to Env Vars

## What Went Well

- Clean migration pattern: `Deno.env.get('X') || fallback` applied consistently across all 12 function references
- Zero test regressions: all 56 tests pass, build succeeds, tsc clean
- Frontend cleaned in two targeted files (Payments.tsx, MetaVerify.tsx) — no longer sends `meta_access_token` in request bodies
- Fallback-first approach eliminates deployment risk: functions continue to work via app_config until env vars are set

## What Could Improve

- MetaCapi.tsx still reads `meta_access_token` from app_config for its settings UI — this is expected (admin config page) but should be reviewed when secrets are finally removed from app_config
- Bundle size warning persists (1,785 kB) — unrelated to this story but worth tracking
