# S10.1 Design: GitHub Actions CI Workflow

## Workflow Structure

File: `.github/workflows/ci.yml`

```yaml
name: CI
on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

jobs:
  ci:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: '20'
          cache: 'npm'
      - uses: denoland/setup-deno@v2
        with:
          deno-version: v2.x
      - run: npm ci
      - name: TypeScript check
        run: npx tsc --noEmit
      - name: Frontend tests
        run: npx vitest run
      - name: Backend tests
        run: deno test supabase/functions/_tests/ --allow-env --allow-net=none
      - name: Build
        run: npm run build
```

Placement: top of README.md, before heading.

## Decisions

- **npm ci** over npm install for reproducible builds
- **Single job** — no parallelism needed for S-size CI
- **Node 20 LTS** — matches local dev environment
- **Deno v2.x** — matches local deno version
- **npm cache** via setup-node built-in cache option
