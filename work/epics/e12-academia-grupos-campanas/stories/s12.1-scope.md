# S12.1: Scope — Schema BD + Edge Functions base

## In Scope

- Migration SQL: columnas en `courses` + tabla `contact_whatsapp_groups` + RLS + índices
- Edge function `list-whatsapp-groups`: proxy a GOWA `/user/my/groups`
- Edge function `sync-group-members`: GOWA `/group/participants` → cruce con contacts → upsert junction
- Edge function `send-group-message`: envío a group JID vía GOWA `/send/message`
- Normalización de teléfonos México (521 vs 52)

## Out of Scope

- UI (viene en S12.2)
- Migración de Campañas a Academia (viene en S12.3)
- Templates de campaña para grupos
- Media en mensajes de grupo (solo texto v1)
