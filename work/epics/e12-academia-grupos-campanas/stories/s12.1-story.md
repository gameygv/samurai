# S12.1: Schema BD + Edge Functions base

## User Story (Connextra)

**Como** agente de ventas
**Quiero** que el sistema tenga la infraestructura para vincular grupos de WhatsApp a cursos y sincronizar miembros
**Para** poder después vincular grupos y enviar campañas desde la UI

## Acceptance Criteria (Gherkin)

### AC1: Migration BD aplicada
```gherkin
Given la base de datos actual
When se aplica la migration
Then courses tiene columnas whatsapp_group_jid y whatsapp_channel_id
And existe la tabla contact_whatsapp_groups con RLS habilitado
```

### AC2: list-whatsapp-groups funciona
```gherkin
Given un canal GOWA activo (edith o anahi)
When llamo a la edge function list-whatsapp-groups con el channel_id
Then retorna la lista de grupos con jid, nombre y conteo de participantes
```

### AC3: sync-group-members funciona
```gherkin
Given un grupo de WhatsApp con miembros
When llamo a sync-group-members con channel_id y group_jid
Then obtiene los participantes de GOWA
And cruza los teléfonos con la tabla contacts
And inserta/actualiza registros en contact_whatsapp_groups
And retorna conteo de synced, matched y unmatched
```

### AC4: send-group-message funciona
```gherkin
Given un grupo de WhatsApp vinculado a un curso
When llamo a send-group-message con channel_id, group_jid y message
Then envía el mensaje al grupo vía GOWA API
And retorna confirmación de envío
```
