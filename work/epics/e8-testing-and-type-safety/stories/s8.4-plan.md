# S8.4: Playwright e2e — Plan

## Tasks

### T1: Install Playwright + Chromium browser
- `npm install -D @playwright/test`
- `npx playwright install chromium`
- Verify: `npx playwright --version` outputs version
- Commit

### T2: Create playwright.config.ts
- baseURL: `http://localhost:8080`
- testDir: `e2e/`
- webServer: `npm run dev` on port 8080, reuseExistingServer for local
- Projects: `setup` (auth.setup.ts) and `chromium` (depends on setup)
- Chromium project uses storageState from `e2e/.auth/user.json`
- Add `e2e/.auth/` and `.env.test` to `.gitignore`
- Create empty `e2e/` directory
- Commit

### T3: Create e2e/auth.setup.ts
- Read `E2E_USER_EMAIL` and `E2E_USER_PASSWORD` from env
- Navigate to `/login`
- Fill email input (`#email`), password input (`#password`)
- Click submit button ("Acceder al Sistema")
- Wait for URL to be `/` (dashboard loaded)
- Save storage state to `e2e/.auth/user.json`
- Commit

### T4: Create e2e/login.spec.ts
- Test: valid login — fill creds, submit, verify navigates to `/`
- Test: invalid login — wrong password, verify error toast visible
- Test: redirect if authenticated — use saved auth state, go to `/login`, verify redirected to `/`
- These tests do NOT use the global auth setup (they test login itself)
- Commit

### T5: Create e2e/dashboard.spec.ts
- Uses saved auth state (authenticated)
- Test: dashboard loads — navigate to `/`, verify heading or key element
- Test: stat cards visible — verify stat card containers are rendered
- Commit

### T6: Create e2e/navigation.spec.ts
- Uses saved auth state
- Test: navigate to Contacts — go to `/contacts`, verify page loads
- Test: navigate to Leads — go to `/leads`, verify page loads
- Test: navigate to Settings — go to `/settings`, verify page loads
- Commit

### T7: Create e2e/contacts.spec.ts
- Uses saved auth state
- Test: contacts list renders — navigate to `/contacts`, verify list/table visible
- Test: click contact opens chat — click first contact row, verify chat panel
- Commit

### T8: Add test:e2e script to package.json
- Add `"test:e2e": "playwright test"` to scripts
- Commit

### T9: Verify all gates pass
- `npx playwright test` — all e2e tests pass
- `npm test` — vitest still passes
- `npm run build` — production build still works
- Commit (if any fixes needed)
