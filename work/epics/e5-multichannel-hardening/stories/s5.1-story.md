# S5.1: Procesador de Recordatorios por Lead

**As** a sales agent
**I want** the reminders I create in the chat to be sent to me via WhatsApp
**So that** I don't forget to follow up with leads at the right time

## Context

The UI already exists (RemindersBlock + ReminderItem in MemoryPanel). Reminders are
stored in leads.reminders JSONB. Each reminder has: id, title, datetime, notify_minutes,
notify_wa. What's missing is the processor that checks and sends them.

## AC

Given a reminder with notify_wa=true and datetime="2026-04-01T10:00" and notify_minutes=15
When the cron runs at 09:45 or later
Then the agent (lead.assigned_to) receives a WhatsApp message via default_notification_channel
And the reminder is marked as sent (not re-sent next hour)

Given a reminder with notify_wa=false
When the cron runs
Then no WhatsApp is sent (skip silently)

Given a lead with no assigned_to
When a reminder triggers
Then skip with log warning (no agent to notify)
